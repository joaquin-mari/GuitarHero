GuitarHero

GuitarHero is a chord recognition application built to detect and identify guitar chords in real time.

I built this project mainly to learn more about software deployment and DevOps practices, not just the application itself.

Over the course of this project, I’ve been focusing on things like Dockerization, Docker Compose setups, deploying applications on a VPS environment, setting up CI/CD pipelines with GitHub Actions, and working more comfortably with Linux.

The goal was to understand how a real application moves from local development to a deployed system that can run reliably in a production-like environment.

This project is still evolving as I continue learning and improving both the codebase and the infrastructure around it.
